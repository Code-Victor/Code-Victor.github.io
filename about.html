My Journey

My journey into web development was propelled by my passion to solve problems and build products that could solve day-day human problems.This passion has constantly fueled me through this journey even in times of uncertainties and against all odds I have continued to thrive and learn.

Started learning python and machine learning.

I started learning python in the year 2020 at Data Scientist Network Lagos, where I was taught its application in machine learning. I learned to build a simple text classifier with Scikit-Learn(a python ML library). With these skills, I was able to do python challenges on Hackerrank and solve ML problems like this IMDb rating prediction and a python helper to download youtube videos

Picked Html, CSS, and Javascript.

With the help of Freecodecamp.org and lots of youtube videos, I was able to gain a firm grasp of building user interfaces with Html and CSS. And I began building landing pages like Chirp, the interior, and the team page. After mastering Javascript, I was able to add interactivity to my apps which made it easier to build more complex apps like Crowdfunding product page and Insure landing page

Delved into Reactjs.

Plain javascript couldn't meet the demands of the projects I began to do, so I learned React to make building complex user interfaces easy. And it delivered. here are the results jadoo landing page, keyboarder, and GitHub repo details

Teaching the community python programming.

As an active member of Data scientist Network at my University(OAU), I taught a community of learners how to use python programming for data science and also offered mentorship.

I wrote my first Blog post.

I always add a passion for teaching, so I began a blog post to teach people about tech and productive software development. here is my first post about Layout animations for the web.
